CSS TRANSITION

La proprietà 'transition' in CSS applica una traniszione tra due valori di una proprietà, oppure addirittura di più proprietà insieme

È sostanzialmente possibile rallentare l'applicazione di un nuovo valore per una proprietà, lascinado al browser il compito di interpolare tutti i valori intermedi.

la proprietà 'transition' solitamente si utilizza per l'effetto di hover.
